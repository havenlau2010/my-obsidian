## 5. 费用系统

### 5.1 基础概念

#### 5.1.1 费用和成本

##### 5.1.1.1 成本

> 花出去变为**有形资产**（**花完看得见**），如：土地购置费，三通一平，建安类

##### 5.1.1.2 费用

> 花出去变为**无形消耗**（**花完看不见**），如：营销推广、行政、差旅、招待等

##### 5.1.1.3 区别

> **成本核算**可以体现出企业**生产的收率情况**，**费用核算**体现出企业的**经营管理效率**

##### 5.1.1.4 企业财务组成

| 财务组成 | |  |  |
| ---- | ---- | ---- | ---- |
| 利润 | 费用 | 税费 | 成本 |

##### 5.1.1.5 费用组成

| 营销费用 | 管理费用 | 财务费用  |
| ---- | ---- | ---- |
| 推广类费用 | 工资  | 利息支出 |
| 人员类费用 | 福利  | 银行手续费 |
| 行政类费用 | 办公费用  | 刷卡手续费 |
| ... | 固定资产折旧  | ... |

##### 5.1.2.2 预算模型（管控工具）

1. 费用预算计算公式

   ```项目费用预算目标(6千万) = 项目销售总货值(20亿) * 费率比(3%)```

2. 流程：

   1. 项目启动：全盘预算确定
   2. 逐年分解，年度不超全盘，每年：分解到科目、月度形成年度预算
   3. 逐月分解，月度累计不超年度，每月：分解到事项形成月度营销预算

### 5.2 费用初始化

#### 5.2.1 科目

##### 5.2.1.1 调研基础

1. 有健全的预算体系，从**预算表中提取**后财务确认
2. 无预算体系，与财务确定**管理科目**是否与**财务会计科目**是否一致

##### 5.2.1.2 要点

 1. 管理科目必须与财务科目相等或者更细，***不允许管理科目比财会科目粒度粗***

 2. 当客户有多个其他板块需要同费用系统管理时，可以配置多套模板授权给对应公司，但是需要注意提前梳理分析其他板块的预算、合同、报销业务流程规则是否与系统匹配

    > 多系统对接，以科目作为数据统计口径的话，如果多系统之间的统计口径不一致，后期要做全盘数据分析很难做

 3. 区分开发间接费和管理费，**开发间接费**的费用类型选择**其他**

    > 开发间接费作为费用大类独立存在，不能包含在管理费中

### 5.3 合同管理
##### 概念

1. 有总价合同：合同有明确的总价约定，如：采购合同，营销推广合同，合同约定总价

2. 无总价合同：合同无明确的总价约定，如：快递费用，水电费用，合同约定单价

3. 跨公司分摊合同：集团签订合同，预算分摊到子公司，支付也是子公司法人

4. 合同预付：先预付再签合同，冲销预付款
