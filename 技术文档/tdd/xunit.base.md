# XUnit

## Automated Tests 自动化测试

    1. 为什么要使用自动化测试
        + 可以频繁的进行测试
        + 可以在任何时间进行测试，也可以按计划定时进行，例如：可以在半夜进行自动测试。
        + 肯定比人工测试要快。
        + 可以更快速的发现错误。
        + 基本上是非常可靠的。
        + 测试代码与生产代码紧密结合。
        + 使得开发团队更具有幸福感！
