# Yarn 用法

## 初始化项目

> yarn init

## 添加依赖包

> yarn add [package]
>
> yarn add [package]@[version]
>
> yarn add [package]@[tag]
>
> yarn add [package] [--dev|--per|--optional]

## 升级依赖包

> yarn add [package]
>
> yarn add [package]@[version]
>
> yarn add [package]@[tag]
>
> yarn add [package] [--dev|--per|--optional]

## 移除依赖包

> yarn remove [package]

## 安装项目的全部依赖

> yarn 或者
>
> yarn install

## windows 10 安装 yarn

  + 安装 ```npm install -g yarn```
  + 验证安装 ```yarn --version```
  + 提示无法加载yarn.psl .以管理员模式打开powershell。执行```set-ExecutionPolicy RemoteSigned```
